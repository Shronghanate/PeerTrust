# **App Name**: PeerTrust

## Core Features:

- User Authentication: Secure user login with email/phone and OTP verification.
- Profile Management: Create and manage user profiles with roles and verification badges.
- Interaction Confirmation: Generate and input interaction codes/QR codes for verified peer interactions.
- Feedback Collection: Collect multi-criteria star ratings, strengths, and improvement areas through structured feedback forms.
- Dashboard Summarization: Summarize ratings with insights and improvement suggestions. The LLM will use reasoning to decide how to incorporate feedback into a summary tool.
- Feedback Request: Request feedback from other users; manage and view their feedback requests.
- Settings & Privacy: Control feedback visibility, block users, and report feedback.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a professional and trustworthy feel.
- Accent color: Soft green (#22C55E) to indicate growth and positive feedback.
- Background color: Off-white (#F9FAFA) to maintain a clean and neutral aesthetic.
- Body and headline font: 'Inter', a sans-serif font, for a modern, machined, neutral look.
- Use Material Icons for a consistent and recognizable UI.
- Employ a card-based layout with rounded corners, soft shadows, and clean spacing.
- Use smooth transitions and subtle animations for enhanced user experience.
